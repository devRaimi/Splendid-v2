QUICK REFERENCE

Setup:
  npm install
  npm run dev

Build:
  npm run build

Edit these files to customize:
  ✏️  src/config.js          → Business settings
  ✏️  src/styles.css         → Colors & styling
  ✏️  src/components/*.jsx   → Modify features

Access modes:
  🛍️  Customer: http://localhost:3000 (default)
  🔐 Admin:    Ctrl+A (keyboard shortcut)
  PIN:         1234 (change in config.js)
